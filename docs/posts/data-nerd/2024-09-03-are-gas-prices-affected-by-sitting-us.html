Gas Prices in USA, historical analysis

This report is intended to review gas prices in the USA historically for comparison against various claims.

One such claim is that the sitting US President has a direct affect on gas prices.

Data from the EIA - US Energy Information Administration

This dataframe set GasPrices_eia_prices_1970_2022 comes from the EIA website as a downloadable CSV.

The EIA provides an FAQ for using the data, which includes instructions to download the CSV and for a reference Excel document that helps with conversion.

â€œTo obtain the historical prices from the SEDS data, use the CSV file for All Statesâ€”Prices. In the file, the code for gasoline prices for the transportation sector, in \(/MMBtu</span></span>, is: State Abbreviation (in column A) and <span class="VerbatimChar"><span style="font-size: 11pt;">MGACD</span></span> (in column B). For example, the code for Alaska is <span class="VerbatimChar"><span style="font-size: 11pt;">AKâ€”MGACD</span></span>. Those prices, in\)/MMBtu, can be converted to approximate dollars per gallon using the heat contents in Table A3 Petroleum consumption and fuel ethanol. There are 42 U.S. gallons in a barrel.â€

The CSV is downloaded from > Section: Prices and expenditures > Subsection: All price and expenditure estimates > SubSubsection: Prices > 1970-2022 > XLSX CSV ZIP

The following shows the barrel prices falling sharply well before 2016â€™s national election, and remaining relatively stable until after the pandemic crashes of 2020.
