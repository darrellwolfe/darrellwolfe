Assessor Parcels Demo

This is a smaller, static recreation of the larger DataViz concept: parcel geometry from the county shapefile, metrics from local CSV exports, and a clean split between Information, Maps, Graphs, and Tables.

Information Maps Graphs Tables

Demo Scope

Static files only. No live database connections.

What This Demo Uses

Parcel polygons are derived from the county shapefile and converted into GeoJSON. Parcel attributes and metrics come from the local CSV extracts in assets/data/assessor_data.

How The Pages Split Up

Maps colors parcels by District and highlights AIN or PIN partial matches in bright yellow.

Graphs counts parcels by District and GEO using the same filter state.

Tables shows the parcel rows behind the map and graph views.

Shared Filters

The Maps, Graphs, and Tables pages all share the same District, GEO, AIN, and PIN filters through browser storage so you can move between pages without re-entering them each time.

Sample GEO Groups In This Demo

The demo uses a compact multi-district sample rather than the full county parcel set, so filters and charts stay useful without becoming heavy.
